EDCI 338 - Social Media and Personalized Learning

Latest Build - 2022-03-30

Course Description

EDCI 338 examines the application of social media in educational contexts and its relationship to personalized learning. The course utilizes blogs and other social media tools to develop networked literacy and learning networks that contribute to their professional practice in legal, safe, consensual and ethical ways.

Course Developers

Jesse Miller is the primary author of the materials in this course, and Colin Madland has revised some sections for the Spring 2022 cohort.
